#aws-3tier-migration-portfolio

🟦 プロジェクト概要
本リポジトリは、中小企業向けオンプレミスWebシステムをAWSへ移行することを想定した
3層アーキテクチャ構成のポートフォリオプロジェクトです。

AWSのベストプラクティスに沿って、
スケーラブル・高可用性・セキュアな構成を意識して設計・構築しています。

🟦 含まれているドキュメント
本プロジェクトには以下の資料（日本語／英語）が含まれています：

- アーキテクチャ図
- インフラ設計書（日本語／英語）
- 構築手順書（日本語／英語）
- 検証証跡（スクリーンショット）
- テスト結果報告書（日本語／英語）
- コスト試算書（日本語／英語）
- セキュリティ設定一覧（日本語／英語）
- 改善提案書

🟦 アーキテクチャ構成（3層構造）
本構成は以下のAWSサービスで構成されています：

- ALB（Application Load Balancer）
- EC2（nginx + PHP-FPM）
- RDS（MySQL Multi-AZ）
- S3（静的ファイル・バックアップ）
- CloudWatch / SNS（監視・アラート通知）
- Route53（DNS）
- IAM（認証／権限管理）
- VPC（Public / Private / DBサブネット構成）

🟦 技術スタック
- AWS: EC2, RDS(MySQL), S3, NAT Gateway, Route53, ALB, CloudWatch, SNS
- OS: Amazon Linux 2
- ミドルウェア: Nginx, PHP-FPM
- DB: MySQL 8.0
- その他: シェルスクリプト、ネットワーク基本設定

🟦 セキュリティについて
本リポジトリに含まれる全資料のうち、
AWSアカウントID・ARN・エンドポイント名・IPアドレス・認証情報などの機密情報は
すべてマスキングまたはサンプル値に置き換えています。

安心して閲覧・レビューできる内容です。アーキテクチャ図
